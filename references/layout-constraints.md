# 布局约束

源自 `next-ai-draw-io` 的布局规范，确保图表元素排列整齐、间距一致、易于阅读。

## 画布约束

| 参数 | 值 | 说明 |
|------|-----|------|
| X 范围 | 0-800 | 水平可用空间 |
| Y 范围 | 0-600 | 垂直可用空间 |
| 容器最大宽度 | 700 | 分组/泳道最大宽度 |
| 容器最大高度 | 550 | 分组/泳道最大高度 |
| 起始边距 | x=40, y=40 | 最小偏移量 |

## 网格对齐

所有坐标对齐 **20px 网格**。

```
正确: x=40, 60, 80, 100, 120...
错误: x=43, 67, 89, 105...
```

## 标准形状尺寸

| 形状类型 | 宽度 | 高度 | 用途 |
|---------|------|------|------|
| 流程框 | 120 | 60 | 标准处理步骤 |
| 小框 | 120 | 40 | 简短标签 |
| 决策菱形 | 120 | 80 | 条件判断 |
| 大框 | 160 | 80 | 复杂标签 |
| 圆形/椭圆 | 80 | 60 | 起点/终点 |
| 数据库 | 120 | 80 | 数据存储 |

## 间距规范

| 方向 | 间距 | 说明 |
|------|------|------|
| 水平 (同行) | 150-200px | 为边路由留出通道 |
| 垂直 (同列) | 50-80px | 紧凑排列 |
| 容器内边距 | 20px | 子元素距容器边缘 |
| 标题间距 | 10px | 标题距顶部边缘 |

## 布局模式

### 水平布局 (左→右)

```
[Node A]  ←150px→  [Node B]  ←150px→  [Node C]
   x=40               x=210              x=380
   y=100              y=100              y=100
```

### 垂直布局 (上→下)

```
[Node A]  y=80
    ↓ 60px
[Node B]  y=200
    ↓ 60px
[Node C]  y=320
```

### 网格布局 (2×2)

```
[A] x=40,  y=80     [B] x=210, y=80
[C] x=40,  y=200    [D] x=210, y=200
```

### 中心辐射

```
          [Top]
           x=200, y=60
             ↓
[Left] ← [Center] → [Right]
x=40     x=200       x=360
y=160    y=160       y=160
             ↓
        [Bottom]
         x=200, y=260
```

## 容器尺寸计算

```
容器宽度  = max(子元素总宽度) + 2 × 内边距 (40px)
容器高度  = max(子元素总高度) + 标题高度 (30px) + 2 × 内边距 (40px)
```

## 标题位置

```
标题: x=画布中心-100, y=20, width=200, height=30
```

## 时序图特殊约束

| 元素 | Y 坐标 | 说明 |
|------|--------|------|
| 参与者框 | y=80 | 顶部水平排列 |
| 生命线起点 | y=120 | 参与者框下方 |
| 消息 1 | y=160 | 第一条消息 |
| 消息 N | y=160+(N-1)×50 | 每条消息间隔 50px |
| 生命线终点 | y=消息最后一条+80 | 留出底部空间 |

参与者框 X 坐标：
```
参与者 1: x=40
参与者 2: x=220
参与者 3: x=400
参与者 4: x=600
参与者 5: x=780
```
生命线 X 中心 = 参与者框 X + 参与者框宽度/2
