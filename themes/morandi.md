# morandi — 莫兰迪

源自 FlowForge，适合高级感、低调优雅的设计。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#D8E2EA` | `#92A8BC` | 起点/终点 |
| process | `#E2D8E6` | `#B5A3BC` | 标准处理 |
| accent | `#EDE8E1` | `#C0B5A8` | 关键转折 |
| neutral | `#DDE6DB` | `#9DB599` | 外部系统 |
| success | `#D6E4D8` | `#8AAE8C` | 成功输出 |
| warning | `#E8E2D2` | `#C4B48E` | 决策/分支 |
| error | `#E8D6D6` | `#B89090` | 错误路径 |
| secondary | `#DBD8E6` | `#A098B5` | 辅助步骤 |
| storage | `#D4DFD8` | `#8DA898` | 数据库/存储 |
| group | `#EDEAE6` | `#C5BFB8` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#92A8BC` |
| 控制流 | `#B89090` |
| 数据转换 | `#8AAE8C` |
| 异步/事件 | `#A098B5` |
| 反馈/循环 | `#C4B48E` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.2;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1;endArrow=classic;endFill=1;
```

## 文字样式

| 元素 | 颜色 | 字号 | 字重 |
|------|------|------|------|
| 标题 | `#4A4A4A` | 16 | 600 |
| 节点 | `#4A4A4A` | 13 | 400 |
| 副标题 | `#7A7A7A` | 11 | 400 |
| 箭头标签 | `#B5AFA6` | 10 | 400 |

## 容器样式

```
swimlane;startSize=24;rounded=1;arcSize=8;strokeWidth=1;strokeDashPattern=4 3;html=1;
```
