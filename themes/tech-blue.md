# tech-blue — 科技蓝灰

源自 FlowForge 默认主题，适合技术内容、现代设计。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#E8F0FE` | `#5B8DEF` | 起点/终点 |
| process | `#D6E4F9` | `#5B8DEF` | 标准处理 |
| accent | `#FFF0E5` | `#E8A87C` | 关键转折 |
| neutral | `#EDF2F7` | `#A0AEC0` | 外部系统 |
| success | `#E6F5EC` | `#5BA87C` | 成功输出 |
| warning | `#FFF8E1` | `#D4A843` | 决策/分支 |
| error | `#FDE8E8` | `#D05050` | 错误路径 |
| secondary | `#EDE8F5` | `#8B7EC4` | 辅助步骤 |
| storage | `#E8F4F2` | `#5BA8A0` | 数据库/存储 |
| group | `#F5F7FA` | `#CBD5E0` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#5B8DEF` |
| 控制流 | `#D05050` |
| 数据转换 | `#5BA87C` |
| 异步/事件 | `#8B7EC4` |
| 反馈/循环 | `#E8A87C` |
