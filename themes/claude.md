# claude — Anthropic 风格

源自 fireworks Style 6，适合 Anthropic 风格演示、AI 相关文档。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#faf8f5` | `#d4a574` | 起点/终点 |
| process | `#ffffff` | `#e0d5c9` | 标准处理 |
| accent | `#d4a574` | `#b8864e` | 关键转折 |
| neutral | `#f5f0ea` | `#c8bfb5` | 外部系统 |
| success | `#e8f0e4` | `#8aae6a` | 成功输出 |
| warning | `#f5ecd0` | `#c4a040` | 决策/分支 |
| error | `#f0dcd0` | `#b07060` | 错误路径 |
| secondary | `#e8e0d8` | `#a89088` | 辅助步骤 |
| storage | `#e8e4da` | `#9a9478` | 数据库/存储 |
| group | `#f8f5f0` | `#d8d0c5` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#b8864e` |
| 控制流 | `#c05040` |
| 数据转换 | `#6a8a4a` |
| 异步/事件 | `#8a7060` |
| 反馈/循环 | `#a08060` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.2;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1.2;endArrow=classic;endFill=1;
```

swimlane;startSize=24;rounded=1;arcSize=8;strokeWidth=1;strokeDashPattern=4 3;html=1;
