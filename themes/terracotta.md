# terracotta — 暖陶商务

源自 FlowForge，适合商务、战略内容。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#FAF0E4` | `#D4956A` | 起点/终点 |
| process | `#F0E0CC` | `#B07D4F` | 标准处理 |
| accent | `#E8DDD0` | `#A89279` | 关键转折 |
| neutral | `#F5EDE3` | `#C4A882` | 外部系统 |
| success | `#E8ECDA` | `#8EA06A` | 成功输出 |
| warning | `#F5ECD0` | `#C4A040` | 决策/分支 |
| error | `#F0DCD0` | `#B07060` | 错误路径 |
| secondary | `#E8E0D8` | `#A89088` | 辅助步骤 |
| storage | `#E8E4DA` | `#9A9478` | 数据库/存储 |
| group | `#F5F0E8` | `#D0C4B0` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#B07D4F` |
| 控制流 | `#B07060` |
| 数据转换 | `#8EA06A` |
