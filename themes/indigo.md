# indigo — 靛蓝深邃

源自 FlowForge，适合 bold、权威的设计。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#DBEAFE` | `#3B82F6` | 起点/终点 |
| process | `#DDD6FE` | `#7C3AED` | 标准处理 |
| accent | `#E0E7FF` | `#4F62B0` | 关键转折 |
| neutral | `#E8ECF4` | `#7C8DB5` | 外部系统 |
| success | `#D4F0E0` | `#3AA06A` | 成功输出 |
| warning | `#FEF3C7` | `#D49A20` | 决策/分支 |
| error | `#FEE2E2` | `#DC2626` | 错误路径 |
| secondary | `#E0D8F0` | `#6B50B0` | 辅助步骤 |
| storage | `#D8E8F0` | `#4080A0` | 数据库/存储 |
| group | `#EEF0F8` | `#A0AAC8` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#3B82F6` |
| 控制流 | `#DC2626` |
| 数据转换 | `#3AA06A` |
| 异步/事件 | `#6B50B0` |
| 反馈/循环 | `#7C3AED` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.2;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1;endArrow=classic;endFill=1;
```

| 元素 | 颜色 | 字号 | 字重 |

| 节点 | `#1E293B` | 13 | 400 |
| 副标题 | `#64748B` | 11 | 400 |
| 箭头标签 | `#94A3B8` | 10 | 400 |
