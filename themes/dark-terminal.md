# dark-terminal — 暗黑终端

源自 fireworks Style 2，适合技术博客、开发者文档。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#1e293b` | `#3b82f6` | 起点/终点 |
| process | `#0f172a` | `#475569` | 标准处理 |
| accent | `#7c2d12` | `#ea580c` | 关键转折 |
| neutral | `#1e293b` | `#64748b` | 外部系统 |
| success | `#064e3b` | `#10b981` | 成功输出 |
| warning | `#78350f` | `#f59e0b` | 决策/分支 |
| error | `#7f1d1d` | `#ef4444` | 错误路径 |
| secondary | `#3b0764` | `#8b5cf6` | 辅助步骤 |
| storage | `#134e4a` | `#14b8a6` | 数据库/存储 |
| group | `#1e293b` | `#475569` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#3b82f6` |
| 控制流 | `#ef4444` |
| 数据转换 | `#10b981` |
| 异步/事件 | `#8b5cf6` |
| 反馈/循环 | `#a855f7` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=6;strokeWidth=1.5;fontSize=13;fontColor=#e2e8f0;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1.5;endArrow=classic;endFill=1;
```

## 文字样式

| 元素 | 颜色 | 字号 | 字重 |
|------|------|------|------|
| 标题 | `#f1f5f9` | 16 | 600 |
| 节点 | `#e2e8f0` | 13 | 400 |
| 副标题 | `#94a3b8` | 11 | 400 |
| 箭头标签 | `#94a3b8` | 10 | 400 |

## 容器样式

```
swimlane;startSize=24;rounded=1;arcSize=6;strokeWidth=1;strokeDashPattern=4 3;html=1;fontColor=#f1f5f9;
```
