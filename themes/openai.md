# openai — OpenAI 风格

源自 fireworks Style 7，适合 API 文档、技术报告、极简设计。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#ffffff` | `#10a37f` | 起点/终点 |
| process | `#ffffff` | `#e5e5e5` | 标准处理 |
| accent | `#ffffff` | `#10a37f` | 关键转折 |
| neutral | `#ffffff` | `#e5e5e5` | 外部系统 |
| success | `#f0fdf4` | `#10a37f` | 成功输出 |
| warning | `#fffbeb` | `#f59e0b` | 决策/分支 |
| error | `#fef2f2` | `#ef4444` | 错误路径 |
| secondary | `#fafafa` | `#d4d4d4` | 辅助步骤 |
| storage | `#f8fafc` | `#94a3b8` | 数据库/存储 |
| group | `#ffffff` | `#e5e5e5` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#10a37f` |
| 控制流 | `#f97316` |
| 数据转换 | `#1d4ed8` |
| 异步/事件 | `#71717a` |
| 反馈/循环 | `#10a37f` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.5;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=0;html=1;strokeWidth=1.5;endArrow=classic;endFill=1;
```

## 文字样式

| 元素 | 颜色 | 字号 | 字重 |
|------|------|------|------|
| 标题 | `#0d0d0d` | 16 | 600 |
| 节点 | `#0d0d0d` | 13 | 500 |
| 副标题 | `#6e6e80` | 11 | 400 |
| 箭头标签 | `#6e6e80` | 10 | 400 |

## 容器样式

```
swimlane;startSize=24;rounded=1;arcSize=8;strokeWidth=1;strokeDashPattern=4 3;html=1;
```
