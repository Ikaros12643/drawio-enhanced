# mint — 薄荷清新

源自 FlowForge，适合清新、轻松的设计。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#E0F5F0` | `#6DBFAB` | 起点/终点 |
| process | `#CCF0E8` | `#2A9D8F` | 标准处理 |
| accent | `#FFF4E6` | `#E9A84C` | 关键转折 |
| neutral | `#E8F0ED` | `#8DBFB0` | 外部系统 |
| success | `#D4F0D8` | `#4AAE5C` | 成功输出 |
| warning | `#FFF2D6` | `#D4A040` | 决策/分支 |
| error | `#FCE4E4` | `#C85050` | 错误路径 |
| secondary | `#E0E8F0` | `#7AA0C0` | 辅助步骤 |
| storage | `#D8F0EE` | `#4AADA5` | 数据库/存储 |
| group | `#F0F7F5` | `#B0D0C5` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#2A9D8F` |
| 控制流 | `#C85050` |
| 数据转换 | `#4AAE5C` |
| 异步/事件 | `#7AA0C0` |
| 反馈/循环 | `#E9A84C` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.2;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1;endArrow=classic;endFill=1;
```

## 文字样式

| 元素 | 颜色 | 字号 | 字重 |
|------|------|------|------|
| 标题 | `#1A3A3A` | 16 | 600 |
| 节点 | `#1A3A3A` | 13 | 400 |
| 副标题 | `#5A7A7A` | 11 | 400 |
| 箭头标签 | `#9DC0B7` | 10 | 400 |

## 容器样式

```
swimlane;startSize=24;rounded=1;arcSize=8;strokeWidth=1;strokeDashPattern=4 3;html=1;
```
