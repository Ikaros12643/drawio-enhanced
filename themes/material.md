# material — Material Design

柔和、专业、低噪声的 Material Design 配色，适合正式文档、演示、产品流程和面向非技术受众的图表。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#E3F2FD` | `#1565C0` | 起点/终点/主要元素 |
| process | `#E8F5E9` | `#2E7D32` | 标准处理步骤 |
| accent | `#FFF3E0` | `#EF6C00` | 关键转换/高亮 |
| neutral | `#ECEFF1` | `#607D8B` | 外部系统/普通模块 |
| success | `#E8F5E9` | `#2E7D32` | 成功输出 |
| warning | `#FFF8E1` | `#FFA000` | 决策/提示/风险 |
| error | `#FFEBEE` | `#C62828` | 错误路径/失败状态 |
| secondary | `#F3E5F5` | `#7B1FA2` | 辅助步骤/特殊模块 |
| storage | `#ECEFF1` | `#607D8B` | 数据库/存储 |
| group | `#FAFAFA` | `#B0BEC5` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#1565C0` |
| 控制流 | `#C62828` |
| 数据转换 | `#2E7D32` |
| 异步/事件 | `#7B1FA2` |
| 反馈/循环 | `#EF6C00` |
| 通用 | `#607D8B` |

## 节点样式

```
rounded=1;whiteSpace=wrap;html=1;arcSize=8;strokeWidth=1.3;fontSize=13;
```

## 箭头样式

```
edgeStyle=orthogonalEdgeStyle;rounded=1;html=1;strokeWidth=1.5;endArrow=classic;endFill=1;
```

## 文字样式

| 元素 | 颜色 | 字号 | 字重 |
|------|------|------|------|
| 标题 | `#263238` | 16 | 600 |
| 节点 | `#263238` | 13 | 400 |
| 副标题 | `#546E7A` | 11 | 400 |
| 箭头标签 | `#607D8B` | 10 | 400 |

## 容器样式

```
swimlane;startSize=24;rounded=1;arcSize=8;strokeWidth=1;strokeDashPattern=4 3;html=1;
```
