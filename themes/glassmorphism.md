# glassmorphism — 毛玻璃

源自 fireworks Style 5，适合产品演示、演讲、官网。

## 色板

| 角色 | 填充色 | 边框色 | 用途 |
|------|--------|--------|------|
| primary | `#1e293b` | `#60a5fa` | 起点/终点 |
| process | `#1e293b` | `#475569` | 标准处理 |
| accent | `#7c2d12` | `#fb923c` | 关键转折 |
| neutral | `#1e293b` | `#64748b` | 外部系统 |
| success | `#064e3b` | `#34d399` | 成功输出 |
| warning | `#78350f` | `#fbbf24` | 决策/分支 |
| error | `#7f1d1d` | `#f87171` | 错误路径 |
| secondary | `#3b0764` | `#a78bfa` | 辅助步骤 |
| storage | `#134e4a` | `#2dd4bf` | 数据库/存储 |
| group | `#0f172a` | `#334155` | 分组容器 |

## 箭头颜色

| 流程类型 | 颜色 |
|---------|------|
| 主数据流 | `#60a5fa` |
| 控制流 | `#f87171` |
| 数据转换 | `#34d399` |
| 异步/事件 | `#a78bfa` |
| 反馈/循环 | `#c084fc` |
